# \# tesseroid-spectral-solution-variable-density

# 

# Spherical harmonic expansions for the gravitational field of a tesseroid with \*\*radially variable density\*\*.

# 

# \## Repository structure

# 

# \- \*\*`mcodes/`\*\*  

# &nbsp; MATLAB scripts and functions.

# 

# \- \*\*`data/`\*\*  

# &nbsp; Input datasets required to run the MATLAB codes.

# 

# \## Key files

# 

# \- \*\*`data/SH\_Marine\_Sediment\_2160.gfc`\*\*  

# &nbsp; Spherical-harmonic potential coefficients of global marine sediments with 3D variable density.

# 

# \- \*\*`Gfield-tesseroid.nb`\*\*  

# &nbsp; Mathematica notebook used to compute a reference gravitational field generated by a \*\*homogeneous\*\* tesseroid.  

# &nbsp; The evaluation point is set at the \*\*North Pole\*\*.

# 

# \## Notes

# 

# \- Ensure the `data/` directory is available and correctly referenced in the MATLAB scripts before running.
